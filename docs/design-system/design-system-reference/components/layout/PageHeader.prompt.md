`PageHeader` tops every routed page — title (16/600), optional one-line description, right-aligned actions, 1px divider below.

```jsx
<PageHeader
  title="Payer Management"
  description="Payers, metadata, and standard operating agreements."
  actions={<Button variant="primary">Add payer</Button>}
/>
```

One primary action max. Sentence case. No marketing copy in the description.
